# bc-fracturing-water-analysis
Analysis of hydraulic fracturing water use in British Columbia using publicly available FracFocus (BCER) data, including trends, summary statistics, and spatial mapping.

## How to Explore the Analysis
- Open `notebooks/bc_water_analysis.ipynb` for the full exploratory analysis
- Outputs (figures and maps) are saved in `/outputs`

## Repository Structure
- `data/` – Raw and cleaned BCER fracture data
- `scripts/` – Data cleaning and preprocessing scripts
- `notebooks/` – Exploratory analysis and visualizations
- `outputs/` – Generated plots and spatial maps
